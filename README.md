## Algorithm Visualizers

Each visualizer animates the algorithm step by step, highlighting active elements and showing what happens at each comparison, swap, or traversal decision.

| Algorithm | Category | Time Complexity | Space Complexity |
|---|---|---|---|
| Bubble Sort | Sorting | O(n²) | O(1) |
| Quicksort | Sorting | O(n log n) avg | O(log n) |
| Binary Search | Search | O(log n) | O(1) |
| Dijkstra's Shortest Path | Graph | O((V+E) log V) | O(V) |
